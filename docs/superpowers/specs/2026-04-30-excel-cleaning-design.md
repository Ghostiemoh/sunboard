# Design Spec: Excel Hacks for Messy Data (Content Package)

**Date:** 2026-04-30
**Topic:** Traditional Data Analysis - Excel Data Cleaning
**Platform:** Instagram, TikTok, LinkedIn, X (Twitter)
**Style:** Shadow Sketch (Digital Whiteboard Aesthetic)

## 1. Goal
Create a high-fidelity, educational, and informative content package that establishes the user as an authority in Data Analysis. The primary deliverable is an 8-slide "Shadow Sketch" carousel for visual platforms and optimized text for long-form platforms.

## 2. Visual Design System
- **Background:** `#09090b` (Deep Zinc/Charcoal)
- **Primary Accent:** `#217346` (Excel Green Glow)
- **Secondary Accent:** `#ffffff` (Clean White Typography)
- **Elements:**
  - Annotated "Mock Spreadsheets" with glowing cells.
  - Bold, handwritten-style arrows for a "whiteboard" feel.
  - High-contrast silhouettes for human/icon elements.

## 3. The Carousel (TikTok/Instagram)
| Slide | Title / Hook | Visual Description | Annotation / Callout |
|-------|--------------|--------------------|----------------------|
| 1 | STOP MANUALLY CLEANING DATA! 🛑 | Bold central text, glowing green aura. | 3 Excel Hacks to save 4 hours/week. |
| 2 | The signal-to-noise problem. | Messy table (mixed cases, spaces). | "Dirty data = Bad insights." |
| 3 | Hack 1: The `TRIM()` Magic. | Formula box `=TRIM(A2)` over a messy cell. | "Kills extra spaces instantly." |
| 4 | Hack 2: Flash Fill (`Ctrl + E`). | Splitting a "Full Name" column into "First Name". | "Excel learns your pattern." |
| 5 | Hack 3: Power Query. | High-tech filter icon processing rows. | "The engine for repeatable alpha." |
| 6 | The Workflow. | 1-2-3 path (Import -> Clean -> Load). | "Standardize for speed." |
| 7 | Pro Analyst Tip. | Glowing lightbulb icon. | "Always duplicate your raw data first!" |
| 8 | Build Your Alpha. | Silhouette of analyst with dashboard. | "Drop 'CLEAN' for my template. Follow." |

## 4. Platform Copy (LinkedIn/X)
- **LinkedIn:** Authority-building article focusing on the ROI of data cleaning for business intelligence.
- **X (Twitter):** Rapid-fire thread focusing on "Efficiency Hacks" for the modern data analyst.

## 5. Implementation Approach
- Use **Browser Subagent** to render Slide 1-8 using Tailwind/HTML/CSS for perfect typography and layout.
- Capture high-resolution screenshots.
- Draft platform-specific copy using `writing-master-class` standards.

## 6. Success Criteria
- Images are crisp, legible, and visually premium.
- The tutorial is easy to follow in under 30 seconds (scroll speed).
- The "Shadow Sketch" aesthetic is maintained (dark mode + glowing highlights).
